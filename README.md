This is the open-source code for Coast 2 Coast Country Club webpage (coast-2-coast.cc).
In the serverless branch is a monorepo of both the Express server routes and Vercel serverless APIs.
Additionally, there is a scraping script utilizing Playwright to scrape scores which populate the leaderboard.
Originally, this project used MERN stack and was deployed on AWS EC2 Lambda, but has since been moved to Vercel.

All golf imagery are used under Unsplash licensing. In addition, video assets are used with consent of its creators.
